anton-remini

anton-remini - A powerful CLI tool that enhances image quality using advanced image processing techniques.

anton-remini supports various image formats such as JPG, JPEG, PNG, and WEBP, making it highly versatile. The tool enables you to enhance individual images, search for images in specific directories, and process entire folders of images automatically.

Usage:

Examples:
  • Enhance a single image: 
    hd input.jpg output.jpg
    Enhances 'input.jpg' and saves it as 'output.jpg'.
  • Search for and process an image: 
    hd -S foto.jpg
    Searches for 'foto.jpg' within /storage/emulated/0. If found, the image will be processed.
  • Display version and help: 
    hd --version
    Displays the version of anton-remini. 
    hd --help
    Shows the help message with usage instructions.
Supported Formats:
  • JPG (.jpg)
  • JPEG (.jpeg)
  • PNG (.png)
  • WEBP (.webp)
